Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Defer откладывает выполнение функции до того момента как внешняя функция завершит свое выполнение нормально 
или через панику. Несколько defer выполняются в порядку LIFO(last in first out). 

В первом случае функция использует именованное возвращаемое значение, которое, после срабатывания return , 
увеличивается отложенной функцией. 

Во втором случае , поскольку, возвращаемое значение уже не именованное, отложенная функция увеличивает значение x в своей области видимости, при этом, так как return функции уже сработал, это не окажет влияние на 
внешнее значение.

Вывод:
2
1
```
