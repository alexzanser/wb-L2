Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Будут выведены цифры с 0 по 9, затем произойдет deadlock, поскольку горутина main продолжает 
читать из канала в цикле, до тех пор, пока он не будет закрыт.

```
